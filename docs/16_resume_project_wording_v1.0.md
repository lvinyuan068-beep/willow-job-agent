## 6. 30秒面试说明

Willow Job Agent 是我从个人真实求职问题出发设计的一个本地AI产品。我负责需求拆解、流程设计、证据边界、输出标准、测试验收和迭代决策，代码部分由AI辅助完成。它输入真实JD后，会先判断是否值得投递，再生成定向简历、HR话术、逐题面试答案和短期补齐方案。目前已经完成双模型主备、Tavily联网搜索、本地网页和四层评测，但真实市场样本仍少，下一阶段会根据HR回复、面试邀请和Offer结果继续校准。

## 7. 面试追问：你在项目中的个人贡献是什么

我的核心贡献不是独立写代码，而是把一个模糊的求职需求转化成可以运行和验收的产品系统。我完成了多轮个人需求梳理，确定岗位优先级、城市薪资底线和风险岗位规则；整理个人经历证据和禁止使用的数据；设计五项核心输出、网页信息结构和测试标准；再使用真实CRM和AIGC岗位发现简历及面试答案问题，推动专项模块迭代并重新测试。AI主要帮助我完成TypeScript代码生成、错误修复和重构。

## 8. 面试追问：这个项目产生了什么结果

目前能够确认的结果是产品和测试结果，而不是招聘转化结果。

已经完成本地可运行网页、岗位筛选、双模型故障切换、Tavily联网检索、证据安全、定向简历、逐题面试答案和结果留档。本地规则8/8、模型切换3/3、网页回归7/7、初始真实岗位2/2通过。

由于真实岗位样本和投递反馈仍少，我不会宣称已经提升面试率。下一步会持续记录投递、HR查看、有效回复、面试和Offer，验证产品是否真正提高求职成功率。
