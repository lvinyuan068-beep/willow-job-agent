# 项目章程 v0.1

## 目标

在不暂停真实求职的前提下，建立一个能持续帮助 Willow 筛选岗位、匹配证据、准备投递和复盘结果的个人求职决策助手。

## 三个月期望结果

- 完成至少一个可展示的小型 AI 项目。
- 优先获得 AI 相关岗位的真实面试与录用机会。
- 如果尚未录用，仍保持投递、作品和证据积累不断线。

## 年底兜底

如果到年底仍没有合适岗位，接受能提供现金流、正式劳动关系且不触碰红线的过渡工作；回衢州是更低优先级方案。

## 当前工作分配

- 求职：50%
- 学习：30%
- 项目：20%

该比例按周评估，不要求每天严格相同。

## MVP 范围

包含：JD 解析、硬条件检查、岗位评分、证据匹配、投递材料建议、面试准备、结果复盘。

暂不包含：自动海投、自动代替本人沟通、批量购买 API、复杂多 Agent 系统、无法追溯来源的简历生成。

## 成功标准

- 对同一岗位给出稳定、可解释的判断。
- 不把愿望写成经历，不把目标指标写成实际结果。
- 能明确回答“为什么投／不投、缺什么证据、下一步做什么”。
- 每次真实反馈都能回写规则，而不是只生成一次性答案。
